PainCare m-Health Application

Overview
The PainCare m-Health solution is a web application derived from a mobile application, designed to address the needs of individuals dealing with endometriosis. Our primary focus is to provide comprehensive resources and support to women struggling with this condition. The application offers personalized symptom tracking, educational materials, and a supportive community to empower users in managing their health effectively.

Features
1. Symptom Tracking
- Users can log and track their pain levels and symptoms on a regular basis.
- The application provides a user-friendly interface for easy and efficient data entry.
2. Diagnosis and Statistics
- Utilizes advanced algorithms to analyze user input and generate potential diagnoses.
- Presents statistical insights based on user data to aid in understanding patterns and trends in symptom occurrences.
3. Personalized Blogs and Community
- Users can create and share personal blogs to connect with others facing similar challenges.
- A community feature encourages collaboration and support among users.
4. Feedback Mechanism
- Users can provide feedback on the application's features and usability.
- The feedback loop helps us continually improve the application based on user suggestions.

Technologies Used

Backend
- Java 2 Platform, Enterprise Edition (J2EE): Providing a robust backend infrastructure.
- Servlet and JSP (JavaServer Pages): Handling dynamic content generation and processing.
Frontend
- HTML, CSS, and JavaScript: Creating a responsive and interactive user interface.

Getting Started
To run the PainCare m-Health web application locally, follow these steps:

1 Clone the Repository:

git clone https://github.com/PainCareMHealthRepo.git
cd PainCareMHealth

2 Setup Backend:

- Install a Tomcat server and deploy the backend application.
- Configure the database settings as per the provided documentation.

3 Run Frontend:

Open the index.html file in a web browser or deploy it on a web server.

4 Access the Application:
- Open your web browser and navigate to http://localhost:8080/PainCareMHealth.
